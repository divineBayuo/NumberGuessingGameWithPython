# NumberGuessingGameWithPython
using simple codes in python, this code shows an effort to make a number guessing game
the player opens the exe and a cmd screen appears with a window and a button to Play.
player selects a lower bound and an upper bound, and then the game returns a number of tries the 
player has to guess the game proportional to the size of the bound. 
The code in python is attached, as well the exe file.
